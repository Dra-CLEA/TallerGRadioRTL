1 La Tubería de Datos

A

Random Source (La Fuente)

Genera números aleatorios entre 0 y 256.
Analogía: Imagina que alguien lanza dados de 256 caras constantemente.

B

Unpack K Bits (El Atomizador)

Rompe el Byte (paquete de 8 bits) en bits individuales.
Si llega el número 13 (00001101), este bloque saca 8 elementos uno tras otro: 0,0,0,0,1,1,0,1.

C

Repeat (El Estirador)

Aquí ocurre la magia del tiempo. Un bit solo dura un instante. Para poder transmitirlo, necesitamos que dure más tiempo.
Repetimos cada bit 1056 veces.
Efecto: En vez de un "ping" instantáneo de '1', tenemos un tono sostenido de '1' durante un rato.

D

UChar To Float (El Traductor)

Entrada (Morado): Datos digitales puros (enteros 0 y 1).
Salida (Naranja): Señales analógicas (flotantes 0.0 y 1.0).
Por qué: Los bloques matemáticos siguientes (como el oscilador VCO) necesitan números decimales (floats) para funcionar, no entienden bytes.
